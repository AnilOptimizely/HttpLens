# Lens family for .NET

This is a placeholder package reserving a package ID for the upcoming Lens family of .NET diagnostic tools.

## What is the Lens family?

The Lens family is a collection of local-first, developer-focused diagnostics packages for modern .NET applications.

Planned packages include:

- HttpLens
- JwtLens
- SqlLens
- AsyncLens
- GcLens
- LogLens
- ConfigLens
- CacheLens
- DiLens
- AiLens
- MailLens
- CookieLens
- BlazorLens
- SignalRLens
- MigrationLens

Each package focuses on making one area of runtime behavior visible through lightweight instrumentation and a shared diagnostics dashboard.

## Status

This package is currently a placeholder. Functional releases are planned as the Lens family evolves.
